Tweet2Growl

起動前の手順（アクセスキーの取得と設定）

はじめに

Tweet2Growlでは、アクセスキーの取得工程全般をユーザーが個別に実施する形式を取っています。

これは、アクセスキー流出による（製作者に振りかかる）諸問題を回避するためです。気になる方は「Twitter アクセスキー 流出」といった単語でググってください。

アクセスキーの取得

Twitterを用いるアプリを開発したことがある人向けの説明

Twitter Developersで新規アプリを登録して、API key, API secret, Access token, Access token secretを取得する。

アクセスキーの設定

twitter4j.propertiesをテキストエディタで開く。
それぞれ以下のように編集して保存する。

  • oauth.consumerKey={API key}
    • {API key}の部分を、[API key]テキストボックスの文字列に置き換える
  • oauth.consumerSecret={API secret}
    • {API secret}の部分を、[API secret]テキストボックスの文字列に置き換える
  • oauth.accessToken={Access token}
    • {Access token}の部分を、[Access token]テキストボックスの文字列に置き換える
  • oauth.accessTokenSecret={Access token secret}
    • {Access token secret}の部分を[Access token secret]テキストボックスの文字列に置き換える

indexページに戻る

※ユーザー向けドキュメントを制作した後に実装を行う方式を取っています。 そのため、ドキュメントが存在していても機能が実装されていない場合があります。
