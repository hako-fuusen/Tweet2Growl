Tweet2Growl

Growl (for Windows) との連携を前提とした Twitter クライアント

Tweet2Growl とは

Growl ないし Growl for Windows との連携を前提とした Twitter クライアントです。
タイムラインに流れる諸情報をGrowlに通知する機能のみを提供します。

主な機能

  • タイムラインの内容をGrowlに通知（UserStream対応）
    • つぶやき
    • 公式RT
    • Favorite及びその解除
    • フォロー、アンフォロー
  • 短縮URLの展開(t.co のみ)
  • NGワード機能

動作環境

Windows7 + Growl for Windows + Java 8 で動作確認をしています。
Growl(及びそのクローン)とJVMが提供されているOSであれば、前述の環境以外でも動くと思われます。

注意点

自分が使うために使ったツールだが、もしかしたら他にも使いたい人がいるかもしれない、というスタンスで公開しています。そのため、下記のような自体が発生しえます。

  • 製作者がツールを使わなくなったため、機能追加及びメンテナンスが行われなくなる
  • 製作者が必要ないと判断した場合、各種要望やバグ報告が無視ないし無言で却下される

※ユーザー向けドキュメントを制作した後に実装を行う方式を取っています。 そのため、ドキュメントが存在していても機能が実装されていない場合があります。
